# Tiny Platform 模块扩展待办清单

## 项目现状

已实现功能：
- 用户/角色/菜单/部门管理 (RBAC权限体系)
- 数据权限控制 (5种权限范围)
- 字典管理
- 登录日志/操作日志
- 文件存储 (本地/MinIO/OSS/S3/COS)
- 代码生成器

---

## 待添加模块

### 高优先级

- [ ] **系统参数配置模块**
  - 全局参数配置
  - 参数分组管理
  - 参数缓存与动态刷新
  - 复杂度：低

- [ ] **通知公告模块**
  - 公告发布与管理
  - 公告类型分类
  - 已读/未读状态跟踪

- [ ] **定时任务模块** (tiny-job)
  - 支持Quartz或XXL-JOB
  - 任务配置管理
  - 执行日志记录

- [ ] **消息通知模块** (tiny-message)
  - 站内消息
  - 邮件通知
  - 短信通知
  - 消息模板管理
  - 消息推送记录
  - 复杂度：中

### 中优先级

- [ ] **数据导入导出模块**
  - 通用Excel导入导出
  - 导入模板管理
  - 异步导出任务
  - 导出记录查询
  - 复杂度：中

- [ ] **系统监控模块**
  - 服务器状态监控 (CPU/内存/磁盘)
  - JVM运行监控
  - Redis缓存监控
  - 数据库连接池监控
  - 复杂度：中

- [ ] **API接口管理模块**
  - 接口文档管理
  - 接口调用统计
  - 接口限流配置
  - API密钥生成与管理
  - 复杂度：中

- [ ] **审计日志增强**
  - 数据变更审计 (记录字段级变化)
  - 敏感操作追踪
  - 审计报告导出
  - 复杂度：中

### 低优先级

- [ ] **内容管理模块** (CMS)
  - 文章管理

  - 富文本编辑器

---

## 推荐实现顺序

```
第一阶段: 系统参数配置 -> 通知公告 -> 数据导入导出
第二阶段: 定时任务 -> 消息通知 -> 系统监控
第三阶段: API管理 -> 审计增强
第四阶段: 工作流 -> 数据大屏 (按需选择)
```

## 技术选型建议

| 模块 | 推荐技术 |
|------|---------|
| 定时任务 | Quartz / XXL-JOB |
| 消息通知 | Spring Mail + 阿里云短信 |
| 数据导出 | EasyExcel |
| 系统监控 | Spring Boot Actuator + Micrometer |
| 工作流 | Flowable |
| 数据可视化 | ECharts |
| 富文本 | TinyMCE / WangEditor |
